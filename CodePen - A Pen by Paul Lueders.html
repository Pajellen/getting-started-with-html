Web Basics

The Internet has multiple elements, the Web being the most visible one to the public eye. The web is a huge interwoven web of servers and cilents constantly communicating with eachother using the HTTP protocol or Hypertext Transfer Protocol expanded out, to create the largest network in the world. When a client is on the web, they are sending HTTP requests to serves to receive documents and data. Most of that data is presented in a web language called HTML, which is the key language we will be learning in this lesson.

The video linked here will provide more info.

HTML

HTML is a very useful language for the Web, it provides a very simple and easy to learn computer-readable language that can allow the meshing of Text and Multimedia content with markup tags that can change the look of the website. This video provides a more indepth review of HTML

Tags and Elements

HTML documents are made up of various elements, which are expressed with tags to manipulate the view of the content created by the creator. A few examples of HTML Tags are:

  • b Tag = Bold, inline
  • em Tag = Emphasis (italics), inline Lack of end tag will make entire text after opening tag be applied Attributes in tags usually are values
  • a Tag = Anchor (usually a link), inline
    • href attribute = specifies URL
  • img Tag = Images, inline
    • src = URL
    • alt = Text (required, shows when you mouse over link)
    • Void Tag, no closing tag needed
  • br Tag = Line break (Void Tag), inline
  • p Tag = Paragraph (Requires end tag), block

Inline or Block elements

Inline = Tag that keeps in line with document
Block = Tag that makes an invisible box with internal formatting

Computer Strengths and limitations

Computers have various strengths and weaknesses, one major strength to computers is their unerring and unflinching nature to follow instructions. However this exact strength can be a weakness as well, computers follow instructions even if those instructions are clearly wrong or misinterpreted. Programmers are challenged with this black and white dogma each time they try to develop code to be used, but with a little finess it can be harnessesed to its great potential
